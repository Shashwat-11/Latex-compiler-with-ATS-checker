# Overleaf LaTeX Editor

A modern, collaborative LaTeX editor built with TypeScript, Turborepo, and pnpm.

## Features

- Real-time LaTeX compilation with Docker sandbox
- PDF preview with pdfjs-dist
- CodeMirror 6 editor with LaTeX syntax highlighting
- ATS resume checker with keyword analysis
- Dark-first theme system with customizable design tokens
- JWT-based authentication with argon2id password hashing

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, CodeMirror 6
- **Backend**: Express, TypeScript
- **Database**: PostgreSQL with Drizzle ORM
- **Build**: Turborepo + pnpm workspaces

## Getting Started

```bash
pnpm install
pnpm dev
```
